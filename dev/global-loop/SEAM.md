   за кампанию, были опасные директивы).
4. **Анти-дублирование SSOT:** архитектура/инварианты/roadmap пульта — ТОЛЬКО
   `factory-conductor/CONDUCTOR.md`; решения — журнал DEC-DEV-0216; хартии — TRACK.md
   треков; порядок работ — PLAN.md. Вторых копий не заводить.
5. **DoD — ссылочный агрегатор** существующих вердиктов (P6/P7/Epic E/graduation/
   impl-sync/DA-RL), НЕ новый судья и НЕ новый артефакт-тип; терминатор цикла = зелёный
   DoD + owner ratify.
6. **H2 gated на events.jsonl** — Q&A-автопилот по pane-peek запрещён (канон «hook-relay,
   не pane-peek»). **R3 gated на реальный prod-хост** — не начинать.
7. **Пре-регистрация перед live-прогонами** ($/sweep и stop-критерии для H0 — до прогона,
   не после); не жечь лимиты (negative exemplar: сессия 100+ агентов).
8. **Знание экосистемы — резолвером, не пересказом** (K0, `CONDUCTOR.md` §Knowledge plane):

   **version-skew** (про проект — из его `.claude/`, про канон — из host-клона);

## Грабли среды (вскрылись в этой сессии)

- **Verify в свежем worktree без `node_modules` → ложный STALE каталога команд**
  (`gen-command-catalog` молча падает на regex-fallback без js-yaml). Лечение:
  `NODE_PATH=<корень>/node_modules npm run verify`. Память: `env-worktree-verify-needs-node-path`.
- `git worktree remove` на Windows иногда «Device or resource busy» — retry `rm -rf` +
  `git worktree prune` через минуту.
- Харнесс памяти нормализует frontmatter записей (обрезает `description` на `#N`,
  оборачивает в кавычки) — Edit по памяти сверяй с фактическим содержимым, не с тем, что писал.
- Номера DEC-DEV раздаёт соседняя сессия тоже — только `npm run next-dec-dev -- --claim`
  (0214/0215 были заняты параллельно).
- main двигается параллельными сессиями несколько раз за день — `git fetch` +
  worktree от `origin/main` перед КАЖДОЙ единицей.
- Упоминание `DEC-DEV-N` в сообщении коммита требует правку DEV_JOURNAL в ТОМ ЖЕ коммите
  (process-gate) — в чисто структурных коммитах номер не упоминать.
- gh-токен на VM невалиден (унаследовано из кампании) — H0-sweep по ssh не требует gh,
  но проверить до H1.
