# Tier-P inbox — проприетарные наблюдения владельца

Сюда — markdown-файлы со своими наблюдениями (боли из разговоров, домен, инсайты).
Свободная форма: 1 наблюдение = 1 абзац, начинай с даты. Пример:

> 2026-07-11: Знакомый тимлид жалуется, что после ухода <вендор X> команда ведёт
> <процесс Y> в Excel — готовы платить за локальный инструмент.

Это единственный вход воронки, который не демократизирован ИИ (CONCEPT.md §3 Tier-P) —
самый ценный. Подмешивается в каждый скан с высоким приоритетом.
