Эта сессия объявлена владельцем **сессией-кондуктором** (ассист-пульт, Волна 0 —
`dev/global-loop/PLAN.md`). Пак собран при вызове команды, результат сборки ниже
(полный пак в файл — вывод в промпт обрезался бы на объёме):

!`node dev/meta-improvement/scripts/host-onboard.cjs --out`

Твоя задача:

1. Прочитай файл пака (путь — в строке `ОНБОРДИНГ-ПАК ЗАПИСАН:` выше) инструментом
   Read **целиком**; если строк больше 2000 — дочитай offset-вызовами. Это
   обязательное первое действие, до любой другой работы.
2. Если в паке есть warn-строки `⚠ НЕ НАЙДЕН` — учти и покажи их владельцу.
3. После прочтения кратко подтверди готовность: секции пака, ~токены, ненайденные
   файлы, и что дальше работаешь по трём законам из шапки пака (SSOT-указатель в
   каждом ответе · version-skew · статус верифицируй git'ом, не паком).
